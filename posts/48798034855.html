would i be right in assuming you can not mix video and stills in a single post with soem text on tumblr?

You can certainly do that!  There are a number a ways you could pull off a post with text, video and and image in the same post.

  1. Create a Text post and use the “upload image” icon on the format bar to insert an image.  Then click on the HTML icon to paste in the embed code for the video.
  2. Or you could create a Video or Photo post and add additional images and videos in the caption using the image and HTML buttons on the format bar.
