Was it fairly easy to make your tumblr site responsive? Did you write a post about this mod?

For the most part all of our Style Hatch themes are fully responsive.  The theme I’m using right now is Cadence, which flows quite well from high resolutions to mobile views.

Personally, when I am building a responsive site I use several media queries to step down the layout in stages.  Once the site width gets into the 600px to 500px range I usually switch to a 100% width fluid layout.  Take a look at the responsive.css file for Cadence to get an idea of how we’re handling the responsive layout.

If you’re using a Tumblr theme that has a responsive layout, make sure you go into the Customize menu and under the Advanced tab uncheck the "Use optimized mobile layout" option.  That will turn off Tumblr’s default mobile theme so that you can see your blog’s theme on a mobile device.
