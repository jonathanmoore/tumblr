MARS—Columns Layout

The latest Style Hatch theme, MARS, to launch last Friday includes three layout modes—default, wide and columns.  To highlight the columns layout, I setup a new MARS columns layout demo site.  The new demo site also shows the flexibility of the theme with different post-type colors, background, and Typekit fonts.  See the original MARS demo site.
